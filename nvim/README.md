# Nvim

```sh
nvim -u init.lua init.lua
```

## Commands

### Explore

Files explorer

- `:Explore` or `:e .`: open file explore
- `%`: when on `:Explore`, creates a new file
- `:find`: when on `:Explore` search a file

### Functioning

- `:healthcheck`

### Config

- `:so`: source init.lua file

### Process

- `CTRL + Z`: Suspend process
- `fg`: back to process
