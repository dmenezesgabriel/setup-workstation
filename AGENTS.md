# Project Guidelines

This is a repository containing several bash scripts and config files mostly for setup linux environment for software development

## Code Style

- Always respect **single responsibility** and **open closed** principles
- Code should be simple, secure, extendable, maintainable, semantic and easy to read
- Cyclomatic complexity must be low

## Code Quality

- `sheelcheck` must be available, you can check it's documentation on how to use it [docs](https://github.com/koalaman/shellcheck)
